# generators

Geradores Node.js que atualizam dados do portfólio/README.

- `certificates.js`: processa certificados.
- `courses.js`: processa cursos.
- `current-book.js`: atualiza leitura atual.
- `current-game.js`: atualiza jogo atual.
- `custom-stats.js`: atualiza estatísticas customizadas.
- `weekly-song.js`: atualiza música semanal.
- `changelog.js`: auxilia na geração de changelog.

Execução sugerida:

```bash
npm run generate
```

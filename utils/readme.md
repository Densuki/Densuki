# utils

Utilitários Node.js compartilhados pelos geradores e scripts.

- `age.js`: cálculo de idade.
- `dates.js`: formatação/cálculo de datas.
- `file.js`: operações de arquivo.
- `github.js`: chamadas/dados relacionados ao GitHub.
- `logger.js`: logs padronizados.
- `markdown.js`: helpers de Markdown.
- `replace.js`: substituição de blocos/conteúdos.

Regras específicas de uma seção devem ficar em `sections/`; utilitários genéricos ficam aqui.

# sections

Seções usadas na geração do README/perfil.

- `profile.js`: dados de apresentação.
- `badges.js`: badges/selos.
- `projects.js`: projetos destacados.
- `skills.js`: habilidades.
- `socials.js`: links sociais.
- `statistics.js`: estatísticas.
- `footer.js`: rodapé do README.

Cada arquivo deve exportar somente a seção pela qual é responsável.

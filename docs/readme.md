
## Manutenção de mídia

A biblioteca de mídia fica em `docs/data/media-library.json`. Inclua novos áudios, imagens e vídeos nos arrays `audio`, `images` e `videos`; o front normaliza esses itens via `docs/js/media-library.js`.

## Frameworks de interface

As páginas HTML carregam Tailwind CDN, Bootstrap 5 e anime.js para facilitar incrementos visuais, componentes responsivos e animações leves.

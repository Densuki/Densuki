# backend/api/debug

Scripts manuais de diagnóstico para uso local.

- `test_db.py`: verifica conexão/modelos do banco.
- `test_health.py`: testa endpoints de saúde.
- `test_login.py`: fluxo básico de autenticação.
- `wake_up.py`: chamada simples para acordar ambientes hospedados.

Esses scripts não substituem testes automatizados, mas ajudam em depuração rápida.
